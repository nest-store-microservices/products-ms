## DEV

# Clonar repositorio

```bash
git clone
```

# Instalar dependencias

```bash
npm install
```

# Crear archivo .env

```bash
cp .env.example .env
```

# Ejecutar migracion de prisma 
```bash
  npx prisma migrate dev --name init
```

# Ejecutar el servidor de desarrollo

```bash
npm run start:dev
```
